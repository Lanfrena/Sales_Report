### 1. Цель проекта
Разработка интерактивной системы сквозного мониторинга коммерческих показателей компании. Проект направлен на визуализацию объемов продаж и заказов в разрезе географии сбыта (страны-реселлеры), товарных категорий и подкатегорий, а также на проведение глубокого временного анализа для сравнения плановых и фактических дат исполнения обязательств.

### 2. Стек технологий
* **Power BI Desktop**
* **Power Query**: ETL данных, очистка и профилирование, построение иерархий категорий товаров.
* **DAX**: Создание кастомных мер для расчета финансовых KPI и управление неактивными связями между таблицами.

### 3. Архитектура данных и реализация USERELATIONSHIP
В основе модели лежит классическая схема «Звезда», где центральная таблица фактов продаж связана со справочниками. 

Особое внимание уделено работе с временными рядами. Так как таблица фактов содержит две критически важные даты — **Дата заказа (Order Date)** и **Дата поставки (Due Date)**, в модели настроены две связи с единым Мастером-календарем:
1. **Активная связь**: по дате заказа (`Order Date`). Все стандартные фильтры по умолчанию рассчитывают продажи на момент оформления сделки.
2. **Неактивная связь**: по дате поставки (`Due Date`). 

Для одновременного вывода обеих метрик на один график была активирована неактивная связь с помощью функции `USERELATIONSHIP` в DAX.

**Пример реализованных мер:**
* *Продажи по дате заказа (базовая мера через активную связь):*
  ```dax
  Sales Amount = SUM('Sales'[SalesAmount])
  ```
* *Продажи по дате поставки (динамическая активация связи):*
  ```dax
  Sales Amount by Due Date = 
  CALCULATE(
      [Sales Amount],
      USERELATIONSHIP('Sales'[DueDateKey], 'Date'[DateKey])
  )
  ```

### 4. Реализованный функционал и аналитика
* **Анализ временных сдвигов:** Настроена интерактивная линейная диаграмма, сопоставляющая `Sales Amount` и `Sales Amount by Due Date`. Это позволяет менеджменту наглядно видеть кассовые разрывы, задержки в поставках и прогнозировать финансовые поступления по периодам.
* **Географическая сегментация:** Интегрирована карта покрытия (`Order Quantity by Reseller Country`), отражающая объемы заказов по ключевым регионам присутствия (США, Канада, Франция, Великобритания, Германия, Австралия).
* **Категорийный анализ (Top-N):** Реализована детализированная матрица продаж, группирующая выручку по типам контрагентов внутри продуктовых категорий, с расчетом общего итога по компании (80,45 млн).
